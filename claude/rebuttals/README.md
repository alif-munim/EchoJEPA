## Key Principles

1. **Lead with the controlled comparison** (EchoJEPA-L vs EchoMAE-L) — it's bulletproof
2. **Acknowledge confounds honestly** — claim qualification builds credibility
3. **Use linear probe numbers** when challenged on fairness — they're defensible
4. **The clustering pattern is supporting evidence**, not the primary argument (partially probe artifact)
